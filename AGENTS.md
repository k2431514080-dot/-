# Codex Instructions

Use the Office Documents skill in `.codex/skills/office-documents/SKILL.md` when working with Microsoft Office files.

## Office file handling

When the task involves Word, Excel, or PowerPoint files, follow these rules:

- For Word `.docx`: preserve layout, edit carefully, check headings/tables/page layout, and verify before final output.
- For Excel `.xlsx` or `.xls`: preserve formulas, keep raw data separate from calculations, use formulas for derived values, format tables clearly, and check for formula errors.
- For PowerPoint `.pptx`: keep each slide focused, use consistent layout, avoid overcrowding, and verify readability.

For Chinese engineering/business reports, use concise, conclusion-first wording suitable for manager review.
