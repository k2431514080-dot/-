# 公司抽奖程序

这是一个简单的命令行抽奖工具，适合公司年会、团建等场景。

## 功能

- 从名单文件读取参与者（支持 `csv` / `txt`）
- 支持多个奖项和名额配置
- 保证同一轮抽奖中不重复中奖
- 结果输出到终端并保存为 JSON 文件

## 快速开始

### 1) 准备名单文件

CSV 示例（第一列为姓名）：

```csv
name
张三
李四
王五
```

或 TXT（每行一个姓名）：

```txt
张三
李四
王五
```

### 2) 运行抽奖

```bash
python3 lottery.py \
  --participants participants_example.csv \
  --prize 一等奖:1 \
  --prize 二等奖:2 \
  --prize 三等奖:3 \
  --seed 42
```

### 3) 查看结果

- 终端会打印中奖名单
- 同时会生成 `lottery_result.json`

## 参数说明

- `--participants`：参与者名单文件路径（必填）
- `--prize`：奖项配置（必填，可重复），格式为 `奖项名:名额`
- `--seed`：随机种子（可选），用于复现结果
- `--output`：结果文件路径（可选），默认 `lottery_result.json`

## 注意事项

- 若奖项总名额超过参与者总数，程序会报错
- 会自动去重重复姓名（按首次出现保留）
